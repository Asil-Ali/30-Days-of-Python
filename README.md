# 30-Days-of-Python
My daily solutions and exercises for the 30 Days of Python course.

This repository contains my daily solutions for the **30 Days of Python** course by Asabeneh Yetayeh.  
Each day has its own folder with Python exercises and practice code.

## Structure
- `day_2/` → Variables, built-in functions
- `day_3/` → Operators
- …and so on

## How to use
- You can browse each day folder to see my solutions.
- Run the Python files using any Python IDE or interpreter.
